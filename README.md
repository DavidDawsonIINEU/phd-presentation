# Interdisciplinary Design and Media PhD Presentation

Prospective student information session for Northeastern University's CAMD PhD program.

## Deployment

This presentation is designed for GitHub Pages deployment.

1. Push all files to your repository
2. Go to Settings → Pages
3. Set Source to "main" branch, root directory
4. Access at: `https://yourusername.github.io/phd-presentation/`

## Navigation

- **Arrow keys** (← →) to navigate slides
- **Spacebar** to advance
- **Click** anywhere to advance (except on interactive elements)
- **Click/Enter** on interactive elements to reveal content

## Interactive Elements

- Slide 2: Click team photos for bios
- Slide 4: Click center logos for descriptions
- Slide 7: Click milestones for details
- Slide 10: Click questions for answers

## Presenting

Open in full-screen mode (F11 in most browsers) for best experience.
